# このソフトウェアについて #

ValueChangedEvent201606242221は私個人が学習目的で作成したソフトウェアです。

色空間であるHSV、RGBを入力すると相互にUIを更新しあいます。
WPFにてEventの削除と再追加で実装しました。

# 開発環境 #

* Windows XP SP3
* .NET Framework 4.0
* WPF
* C#
* Visual C# 2010 Express
